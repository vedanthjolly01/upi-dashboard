[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://your-app-url.streamlit.app)

## 📊 Overview

Interactive dashboard analyzing India's Unified Payments Interface (UPI) growth from 2016 to 2025.

### Features

- 📈 **Growth Analytics** - Track volume and value trends
- 🔄 **P2P vs P2M** - Person-to-Person vs Merchant payments
- 📱 **App Market Share** - PhonePe, Google Pay, Paytm dominance
- 🗺️ **State Insights** - Geographic adoption patterns
- 🛒 **Merchant Analysis** - Category-wise spending
- 🔒 **Security Dashboard** - Fraud trends and ODR rates
- 👥 **Demographics** - Age-wise adoption analysis
- 🌍 **Global Benchmark** - India vs other countries

### Data Sources

- NPCI Official Statistics
- RBI Annual Reports
- Ministry of Finance
- ACI Worldwide Reports

### Tech Stack

- Python 3.9+
- Streamlit
- Plotly
- Pandas
